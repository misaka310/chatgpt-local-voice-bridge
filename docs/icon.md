# 拡張機能アイコン

公開版の Chrome 拡張アイコンは、汎用的なローカル音声ブリッジ用のマークです。

## 元ファイル

```text
extension/assets/icons/icon.svg
```

## 生成されるPNG

```text
extension/assets/icons/icon16.png
extension/assets/icons/icon32.png
extension/assets/icons/icon48.png
extension/assets/icons/icon128.png
```

## デザイン意図

- 吹き出し: ChatGPT の返答 preview
- 波形バー: ローカル TTS 音声
- 小さな点と弧: ローカル接続・ブリッジ

公開アイコンには、キャラクター絵、VTuber絵、個人の音声提供者に紐づく画像、プロジェクト内だけで使う非公開素材を使いません。

個人用のローカル素材は、Gitで無視されるローカル用パスに置き、公開アイコンには含めません。
